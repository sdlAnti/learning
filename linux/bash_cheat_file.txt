clear - очистит ваш терминал
echo - выводит текст на ваш терминал
pwd - print work directory 
head -n - выводит на терминал первые десять строк файла Вы можете указать количество печатаемых строк с помощью опции -n
tail -n - выводит на терминал последние десять строк файла Вы можете указать количество печатаемых строк с помощью опции -n
wc - показывает количество строк в файле
